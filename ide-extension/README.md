# SetupHub Sync

**SetupHub Sync** allows you to synchronize your VS Code, Cursor, or Windsurf settings, extensions, and themes across multiple devices effortlessly.

Say goodbye to manually reinstalling extensions or copying JSON files. Sync it once, use it everywhere.

## Quick Start

1.  **Install** the extension.
2.  Open the **Command Palette** (`Cmd+Shift+P` / `Ctrl+Shift+P`).
3.  Run **`SetupHub: Login`** to authenticate with your SetupHub account.
4.  Run **`SetupHub: Sync My Setup`** to push your current setup to the cloud.

On your other computer, simply install, login, and sync to download your perfect environment!

## Key Commands

Open the Command Palette and type `SetupHub` to see all available commands:

| Command                             | Description                                                        |
| :---------------------------------- | :----------------------------------------------------------------- |
| `SetupHub: Login`                   | Log in to your SetupHub account.                                   |
| `SetupHub: Sync My Setup`           | Sync your current local settings with the cloud (Upload/Download). |
| `SetupHub: Browse Community Setups` | Discover and install popular developer configurations.             |
| `SetupHub: View My Profile`         | View your synced stats and active profile details.                 |
| `SetupHub: Logout`                  | Sign out of your account on this device.                           |

## Requirements

- VS Code v1.85.0 or higher.
- An active internet connection for syncing.
